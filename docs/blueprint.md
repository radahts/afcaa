# **App Name**: AFCAA 2026

## Core Features:

- Candidate Application Wizard: A multi-step form guiding candidates through the application process, including progress tracking and file upload.
- Jury Evaluation Dashboard: A secure dashboard for jury members to evaluate candidates based on a defined grid, with anonymization features.
- Public Voting System: A 'People's Choice' voting platform using magic-links and captcha to ensure fair voting, with real-time leaderboard updates.
- Partner Analytics Dashboard: A dashboard for partners to track their ROI, including impressions, leads, and signed contracts.
- Admin Back-Office: Comprehensive back-office for managing all aspects of the platform, including CRUD operations, moderation, and data export.
- Barter Code Validation: Allows candidates to enter a barter code to nullify amount on payment, generating invoice at the end of the process
- Badge generation: Share of voting leaderboard generates shareable badge NFT ERC-721 on Polygon (metadata stored IPFS)

## Style Guidelines:

- Primary color: Bright gold (#F39C12) to evoke prestige and excellence.
- Background color: Light gray (#F5F5F5), close to white but with slight variation to avoid overly bright background.
- Accent color: African green (#00A859) for call to actions, indicating growth and opportunity.
- Headline font: 'Poppins' (sans-serif) for a modern and geometric style.
- Body font: 'Noto Sans' (sans-serif) for clear and accessible text.
- Use thin line icons in an 'Afro-contemporary' style.
- Implement micro-interactions with Lottie animations (< 80 KB) to enhance user experience. The button has an 'or' ('gold' in French) hover effect.
